# 工业级芯片检测机项目数据仓库——《智能产品设计》第2组

本仓库用于存放工业级芯片检测机设计开发过程中的项目数据。

## 仓库目标

- 统一管理团队设计开发数据
- 支持标准化建模与复用
- 为后续零部件与整机装配协同提供数据基础

## 数据结构规划

### 1. 标准件库（Standard Parts Library）

用于存放通用标准件数据，例如：

- 紧固件
- 轴承
- 导轨
- 电气标准件

### 2. 零件库（Parts Repository）

用于存放项目自研零件数据，例如：

- 机械零件模型
- 加工图与工程图
- 版本与变更记录

### 3. 装配体库（Assembly Repository）

用于存放装配体与系统级数据，例如：

- 子装配体
- 总装模型
- 装配约束与BOM关联数据
